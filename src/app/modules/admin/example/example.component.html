Starter - Example component

This is the Starter version of the Assembly.

While it doesn't include any of the example applications or pages from the Demo app, it still has the essential parts such as Authentication pages, Notifications, Messages and Shortcuts panels, Search component and more.

Build your application on top of this version. You can copy/paste modules from the Demo app into here as you need them. Since Demo app has lots of example applications, sample and informational pages, it also has a lot of weight to it which will slow down the development.
